# Один заказ Ice - 10 шагов
1. Лид пришёл (форма/звонок/GBP) -> 8 вопросов интейка (скрипты: docs/business/scripts.md) сразу.
2. Котировка флэт в одном сообщении. Молчат 24ч -> один follow-up.
3. Депозит 50% (Zelle / Stripe-ссылка). Нет депозита = нет брони.
4. Звонок бартендеру по тиру (скрипты: docs/business/scripts.md). Принял -> Event Order текстом.
5. Лист заказа в трекер (трекер: docs/business/trackers/ice-dispatch.csv) в момент брони.
6. За день: подтверждение бартендеру И клиенту одним сообщением каждому.
7. День события: бартендер на месте за 45 мин -> фото сетапа в WhatsApp.
8. После: остаток оплаты от клиента; Zelle бартендеру в тот же вечер.
9. +24ч: сообщение клиенту - спасибо + ссылка на отзыв (GBP review link) (ссылку на отзыв взять из TEMO-HOUR после настройки GBP; пока её нет - просто тёплое спасибо + просьба рекомендовать друзьям).
10. Закрыть строку в трекере: записать спред. Спред < $200 -> разбор, что
    пошло не так с ценой.
