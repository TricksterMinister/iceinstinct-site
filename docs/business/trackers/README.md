CSV-файлы здесь - шаблоны; импортировать в Google Sheets как "Ice Bench" и "Ice Dispatch".

## Импорт (по шагам)
1. sheets.new -> назвать книгу "Ice & Instinct - Ops".
2. File -> Import -> Upload -> `ice-dispatch.csv` -> "Insert new sheet(s)" -> Import. Переименовать вкладку "Ice Dispatch".
3. Повторить для `ice-bench.csv` -> вкладка "Ice Bench".

## Ice Dispatch - авто-спред и итоги месяца
- Столбцы: A Дата, B Клиент, C Канал, D Город, E Пакет, F Цена клиенту, G Депозит, H Бартендер, I Выплата, J Выплачено, K Спред, L Отзыв запрошен, M Отзыв получен, N Заметки.
- Спред (K2, протянуть вниз): `=IF(F2="","",F2-I2)` (цена клиенту минус выплата).
- Итоги месяца (любые свободные ячейки): сумма спреда `=SUM(K2:K)`; число заказов `=COUNTA(A2:A)-1`; средний спред `=AVERAGE(K2:K)`.

Минимум спреда = $200/заказ (бизнес-правило). Цифра ниже = пересмотреть цену клиенту или тир бартендера.
